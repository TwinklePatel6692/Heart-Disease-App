❤️ Heart Disease Prediction App

## 📌 Overview
This project is a Machine Learning-based web application that predicts the likelihood of heart disease based on user medical inputs. It helps in early detection and risk assessment using data-driven techniques.


## 🎯 Problem Statement
Heart disease is one of the leading causes of death worldwide. Early prediction can help in timely diagnosis and prevention.

This project aims to build a predictive system that estimates the probability of heart disease using patient health parameters.


## 🚀 Solution
Developed a classification model using Machine Learning algorithms to analyze patient data and predict heart disease risk.

The model is integrated into an interactive web application built with Streamlit, allowing users to input health data and receive instant predictions.

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Visualization:** Matplotlib / Seaborn  
- **Deployment/UI:** Streamlit  
- **Model Saving:** Joblib  


##  Features
- User-friendly web interface  
- Real-time heart disease prediction  
- Input-based risk assessment  
- Data preprocessing and model training  
- Fast and lightweight application

## 📊 Model Details
- Algorithm Used: (Decision Tree/Logistic Regression / Random Forest / SVM /Naive Bayes/KNN)
- we use the one with higher accuracy -KNN
- Evaluation Metrics:
  - Accuracy: 86%  
  - F1-Score: 88% 

## 📥 Input Parameters
The model uses the following health indicators:
- Age  
- Sex  
- Chest Pain Type  
- Resting Blood Pressure  
- Cholesterol Level  
- Fasting Blood Sugar  
- Resting ECG  
- Maximum Heart Rate Achieved  
- Exercise-Induced Angina  
- ST Depression (Oldpeak)
